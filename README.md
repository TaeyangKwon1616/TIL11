#### 논리 조건식과 연속된 if문으로 바꾼 코드
'''c
if ((a>10)&&(b>=0))
{
    b = 1;
}
if ((a>10)&&(b<0>))
{
    b = -1;
} 
printf("%d\n", b);

return 0;
'''
#### 논리곱(&&) 연산은 두 조건을 모두 만족해야 하므로 a>10 조건이 거짓이면 두 if문의 조건식은 모두 거짓이 되어 b값은 바뀌지 않습니다. 따라서 a>10조건이 참일 때만 b의 값에 따라 둘 중 한 문장을 실행하게 됩니다.
